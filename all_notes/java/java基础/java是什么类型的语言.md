##### java既是编译型语言又是解释型语言
java文件会先被编译器为字节码,注意不是机器码
然后将字节码给jvm解释运行
当然jvm会为了性能将一部分字节码一次性编译成机器码给cpu
然后实时解释没有编译的字节码成机器码给cpu
所以说java既是编译型语言又是解释型语言

##### java为什么能跨平台?
就是因为他没有直接被编译为机器码
因为每个硬件平台不同的cpu其指令集不同
指令集不同你相同的机器码会被执行出不同的结果
所以有一个jvm来根据你cpu的不同指令集
来将字节码解释成不同的机器码来兼容已达到跨平台的效果
